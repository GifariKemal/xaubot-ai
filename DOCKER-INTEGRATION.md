## ğŸ› Troubleshooting

```cmd

```

```cmd

```

**Check database:**
```cmd

```

**Check network:**
```cmd
docker network inspect trading_bot_network
```

### Port conflict

Edit `.env` untuk ganti port:
```env
API_PORT=8001
DASHBOARD_PORT=3001
```

Then restart:
```cmd
docker-compose down trading-api dashboard
docker-compose up -d trading-api dashboard
```
